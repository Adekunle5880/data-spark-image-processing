# Data-SPARK Image Processing Project

## 📌 Overview
This project processes images using Python and AI (Gemini API) to generate structured JSON outputs.

---

## ⚙️ Features
- Image analysis using Gemini API
- JSON output generation
- Error handling with retry logic
- Fallback system when API fails

---

## 🏗️ Architecture
Images → Python Script → Gemini API → JSON Output

---

## 📂 Project Structure
- `images/` → input images  
- `outputs/` → generated JSON  
- `main.py` → processing script
